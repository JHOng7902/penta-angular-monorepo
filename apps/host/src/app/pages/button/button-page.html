Button Component

The shared pt-ui-button supports variants, sizes, colors, content modes (text/icon/text-icon), tooltip, and attention animation.

Import & Use

  1. Import the Button component from the shared library.
  2. Add it to the imports array of your standalone component.
  3. Use <pt-ui-button> in the template. 
import { Button } from '@penta/shared-ui';

@Component({
  standalone: true,
  imports: [Button],
})
export class ExampleComponent {}

<pt-ui-button buttonText="Save"></pt-ui-button>

Tip: if you need click handling, use (pressed) or set type="submit" inside a form.

Inputs & Outputs

These are the inputs and outputs supported by the shared pt-ui-button component.

Inputs

  • variant: primary | outline | ghost
  • size: sm | md | lg | xl | xxl
  • content: text | icon | text-icon
  • icon: Material icon name (string)
  • buttonText: text label (string)
  • color: primary | accent | warn | success | info | neutral | danger | fail
  • blink: boolean
  • disabled: boolean
  • type: button | submit | reset
  • ariaLabel: string
  • tooltip: string
  • tooltipPosition: above | below | left | right | before | after | start | end

Outputs

  • pressed: emits MouseEvent on click

Content Modes

Choose between text, icon-only, or text + icon buttons. Use content and icon to control layout. 

<pt-ui-button buttonText="Text" content="text"></pt-ui-button>
<pt-ui-button buttonText="Text + Icon" content="text-icon" icon="bolt"></pt-ui-button>
<pt-ui-button content="icon" icon="settings" ariabuttonText="Settings"></pt-ui-button>

Variants

Variants affect the visual emphasis: primary (filled), outline (border), and ghost (text). 

<pt-ui-button buttonText="Primary" variant="primary"></pt-ui-button>
<pt-ui-button buttonText="Outline" variant="outline"></pt-ui-button>
<pt-ui-button buttonText="Ghost" variant="ghost"></pt-ui-button>

Colors

Custom tones work across all variants. Use success, info, neutral, danger, or warn. 

<pt-ui-button buttonText="Success" color="success"></pt-ui-button>
<pt-ui-button buttonText="Danger" color="danger"></pt-ui-button>

Sizes

Sizes control font and button height. Available: sm, md, lg, xl, xxl. 

<pt-ui-button buttonText="XL" size="xl"></pt-ui-button>
<pt-ui-button buttonText="XXL" size="xxl"></pt-ui-button>

Tooltip + Position

Tooltips appear on hover using tooltip and tooltipPosition. 

<pt-ui-button buttonText="Above" tooltip="Tooltip above" tooltipPosition="above"></pt-ui-button>

Blinking Attention

Use blink for attention states such as alerts or next-step actions. 

<pt-ui-button buttonText="Blink" [blink]="true"></pt-ui-button>

Events

Use (pressed) to handle clicks or set type="submit" in a form to submit.

You can also bind disabled to conditions, for example disabling until an input has text. 

<pt-ui-button buttonText="Save" (pressed)="onSave()"></pt-ui-button>
<form (ngSubmit)="onSubmit()">
  <pt-ui-button type="submit" buttonText="Submit"></pt-ui-button>
</form>

<pt-ui-input type="text" placeholder="Type to enable" (input)="disableName = $any($event.target).value"/>
<pt-ui-button buttonText="Save" content="text-icon" [icon]="'save'" [disabled]="!disableName" size="sm"></pt-ui-button>
