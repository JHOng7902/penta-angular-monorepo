Checkbox Component

The shared pt-ui-checkbox wraps Material's checkbox with consistent styling and helpful inputs/outputs.

Import & Use

  1. Import CheckboxComponent from the shared library.
  2. Add it to the imports array of your standalone component.
  3. Bind using [checked] + (checkedChange).
import { CheckboxComponent } from '@penta/shared-ui';

@Component({
  standalone: true,
  imports: [CheckboxComponent],
})
export class ExampleComponent {}

<pt-ui-checkbox
  label="Accept terms"
  [checked]="accepted"
  (checkedChange)="accepted = $event"
></pt-ui-checkbox>

Basic Usage

A simple checkbox with two-way binding using checked and checkedChange.

Checked: {{ basicChecked }} 
<pt-ui-checkbox
  label="Enable notifications"
  [checked]="basicChecked"
  (checkedChange)="basicChecked = $event"
></pt-ui-checkbox>

Checked + Indeterminate + Disabled

Demonstrates the three common states: checked, indeterminate, and disabled. 

<pt-ui-checkbox label="Checked" [checked]="true"></pt-ui-checkbox>
<pt-ui-checkbox label="Indeterminate" [indeterminate]="true"></pt-ui-checkbox>
<pt-ui-checkbox label="Disabled" [disabled]="true"></pt-ui-checkbox>

Label Position

Switch the label between left (before) and right (after). 

<pt-ui-checkbox
  label="Sample checkbox"
  labelPosition="before"
  [checked]="true"
></pt-ui-checkbox>

Select All Group

Demonstrates checked + indeterminate + disabled items.

@for (item of groupItems; track item.label) { } 
<pt-ui-checkbox
  label="Select All"
  [checked]="allChecked"
  [indeterminate]="allIndeterminate"
  (changed)="toggleAll($event)"
></pt-ui-checkbox>

Extra Options

Configure ripple and required state, then apply them to a checkbox.

Current settings: Ripple {{ disableRipple ? 'Off' : 'On' }} | Required {{ required ? 'On' : 'Off' }} 
<pt-ui-checkbox
  label="Option"
  [disableRipple]="true"
  [required]="true"
></pt-ui-checkbox>

Events

Listen to checkedChange, changed, or pressed.

Last event: {{ lastEvent }} 
<pt-ui-checkbox
  label="Event demo"
  (checkedChange)="onChecked($event)"
  (changed)="onChanged($event)"
  (pressed)="onPressed($event)"
></pt-ui-checkbox>

Customize Styles

Wrap checkboxes in a class and override styles locally. For component-scoped styles, use ::ng-deep or place overrides in a global stylesheet. 

<div class="checkbox-page__custom-theme">
  <pt-ui-checkbox label="Large checkbox" [checked]="true"></pt-ui-checkbox>
  <pt-ui-checkbox label="Accent label"></pt-ui-checkbox>
</div>

:host ::ng-deep .checkbox-page__custom-theme .pt-ui-checkbox {
  color: #0f172a;
}

:host ::ng-deep .checkbox-page__custom-theme .pt-ui-checkbox ::ng-deep .mdc-checkbox {
  transform: scale(1.15);
}

:host ::ng-deep .checkbox-page__custom-theme .pt-ui-checkbox__label {
  color: #ea580c;
}

Inputs & Outputs

Inputs

Name Type Usage
labelstringText shown next to the checkbox.
checkedbooleanControls the checked state.
indeterminatebooleanTri-state visual when partially selected.
disabledbooleanDisable interaction.
requiredbooleanMarks the checkbox as required.
labelPositionbefore | afterLabel placement relative to the box.
namestringNative name attribute.
valuestringNative value attribute.
idstringCustom id attribute.
tabIndexnumberCustom tab index.
disableRipplebooleanDisable Material ripple effect.
ariaLabelstringAccessibility label.
ariaLabelledbystringAccessibility labelled-by reference.

Outputs

Name Type Usage
checkedChangeEventEmitter<boolean>Emits when checked state changes.
changedEventEmitter<MatCheckboxChange>Emits Material change payload.
indeterminateChangedEventEmitter<boolean>Emits when indeterminate changes.
pressedEventEmitter<MouseEvent>Emits on click.
focusedEventEmitter<FocusEvent>Emits on focus.
blurredEventEmitter<FocusEvent>Emits on blur.
