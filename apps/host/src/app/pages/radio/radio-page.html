Radio Component

Use pt-ui-radio for a single yes/no choice. Use pt-ui-radio-group to present multiple options where only one can be selected.

Import & Use

  1. Import RadioComponent and/or RadioGroupComponent from the shared library.
  2. Add them to the imports array of your standalone component.
  3. For single radios, bind [checked] and handle (changed).
  4. For groups, bind [value] / (valueChange) (or [(value)]).
import { RadioComponent, RadioGroupComponent } from '@penta/shared-ui';

@Component({
  standalone: true,
  imports: [RadioComponent],
})
export class ExampleComponent {}

<pt-ui-radio
  label="Primary"
  [checked]="selected"
  (changed)="selected = $event.source.checked"
></pt-ui-radio>

<pt-ui-radio-group
  label="Shift"
  [options]="options"
  [(value)]="selectedValue"
></pt-ui-radio-group>

Basic Usage

A single radio is best for “yes/no” or “enable/disable” choices.

Checked: {{ basicChecked }} 
<pt-ui-radio
  label="Enable priority"
  [checked]="basicChecked"
  (changed)="onBasicChanged($event)"
></pt-ui-radio>

Required Indicator

Set required to show the asterisk and apply native required validation. 

<pt-ui-radio label="Required option" [required]="true"></pt-ui-radio>

Disabled State

Bind disabled to a condition so the control can be locked when needed. 

<pt-ui-radio [disabled]="isLocked"></pt-ui-radio>

Radio Group

pt-ui-radio-group renders a list of options where only one value can be chosen. It accepts an options array, a selected value, and optional disabled rules per option.

Selected: {{ selectedValue }}

Events

Use events to track interaction or update other UI.

Last event: {{ lastEvent }} 
<pt-ui-radio
  label="Event demo"
  (checkedChange)="basicChecked = $event"
  (changed)="onChanged($event)"
  (pressed)="onPressed($event)"
></pt-ui-radio>

Customize Styles

Wrap radios in a class and override styles locally. For component-scoped styles, use ::ng-deep or move overrides to a global stylesheet. 

<div class="radio-page__custom-theme">
  <pt-ui-radio label="Branded Radio"></pt-ui-radio>
</div>

:host ::ng-deep .radio-page__custom-theme .pt-ui-radio ::ng-deep .mdc-radio {
  transform: scale(1.1);
}

:host ::ng-deep .radio-page__custom-theme .pt-ui-radio__label {
  color: #2563eb;
  font-weight: 700;
}

Inputs

Name Type Usage
labelstringText shown next to the radio.
checkedbooleanControls the checked state.
disabledbooleanDisable interaction.
requiredbooleanMarks the radio as required.
labelPositionbefore | afterLabel placement relative to the control.

idstringCustom id attribute.
tabIndexnumberCustom tab index.
disableRipplebooleanDisable Material ripple effect.
ariaLabelstringAccessibility label.
ariaLabelledbystringAccessibility labelled-by reference.
colorprimary | accent | warnMaterial color theme.

Outputs

Name Type Usage
checkedChangeEventEmitter<boolean>Emits when checked state changes.
changedEventEmitter<MatRadioChange>Emits Material change payload.
pressedEventEmitter<MouseEvent>Emits on click.
focusedEventEmitter<FocusEvent>Emits on focus.
blurredEventEmitter<FocusEvent>Emits on blur.
