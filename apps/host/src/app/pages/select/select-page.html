Select Component

The shared pt-ui-select supports single or multiple selection, optional filtering, label positioning, clear action, and disabled options.

Import & Use

  1. Import SelectComponent from the shared library.
  2. Add it to the imports array of your standalone component.
  3. Bind using [(value)] or [value] + (valueChange).
import { SelectComponent } from '@penta/shared-ui';

@Component({
  standalone: true,
  imports: [SelectComponent],
})
export class ExampleComponent {}

<pt-ui-select label="Station" [options]="options" [(value)]="station"></pt-ui-select>

Single Select

Use the default mode for a single selection with a placeholder.

Selected: {{ basicValue || '-' }} 
<pt-ui-select
  label="Basic Select"
  [options]="options"
  [(value)]="basicValue"
></pt-ui-select>

Multiple Select

Enable multiple to select more than one option.

Selected: {{ multiValue.length ? multiValue.join(', ') : '-' }} 
<pt-ui-select
  [options]="options"
  [multiple]="true"
  [(value)]="multiValue"
></pt-ui-select>

Filter Input

Set showFilter to render an input for filtering the option list.

Selected: {{ filterValue || '-' }} 
<pt-ui-select
  [options]="options"
  [showFilter]="true"
  [(value)]="filterValue"
></pt-ui-select>

Option Without Description

Options without description only render the label row.

Selected: {{ basicValue || '-' }} 
// Delta has no description
<pt-ui-select [options]="options"></pt-ui-select>

Label Positions

Control label placement with labelPosition: top, left, or hidden. 

<pt-ui-select label="Left Label" labelPosition="left"></pt-ui-select>
<pt-ui-select labelPosition="hidden" placeholder="No label"></pt-ui-select>

Required Indicator

Set required to display the red asterisk on the label. 

<pt-ui-select label="Required" [required]="true"></pt-ui-select>

Disabled State

Bind disabled to any condition. Toggle below to see it update. 

<pt-ui-select [disabled]="isLocked"></pt-ui-select>

Clear Action

Enable showClear to show a clear icon when a value is selected. 

<pt-ui-select [showClear]="true"></pt-ui-select>

Selected As Chips

Set displayAsChips to render the selected items as chips in the trigger.

Selected: {{ chipValue.length ? chipValue.join(', ') : '-' }} 
<pt-ui-select
  [options]="options"
  [multiple]="true"
  [displayAsChips]="true"
></pt-ui-select>

Disabled Options

Options can be disabled via the option data or a conditional callback. 

<pt-ui-select
  [options]="options"
  [optionDisabled]="optionDisabled"
></pt-ui-select>

Customize Styles

Wrap selects in a class and override styles locally. For component-scoped styles, use ::ng-deep or move overrides to a global stylesheet. 

<div class="select-page__custom-theme">
  <pt-ui-select label="Branded Select" [options]="options" panelClass="select-page__panel-theme"></pt-ui-select>
</div>

:host ::ng-deep .select-page__custom-theme .pt-ui-select__field .mat-mdc-select-trigger {
  border-color: #14b8a6 !important;
  background: #f0fdfa !important;
}

:host ::ng-deep .select-page__custom-theme .pt-ui-select__value {
  color: #0f766e;
}

:host ::ng-deep .select-page__custom-theme .mat-mdc-select-arrow {
  color: #0f766e;
}

:host ::ng-deep .select-page__panel-theme .pt-ui-select__filter-wrap {
  background: #ecfeff;
  border-color: rgba(20, 184, 166, 0.5);
}

:host ::ng-deep .select-page__panel-theme .mat-mdc-option.mat-mdc-option-selected {
  background: rgba(20, 184, 166, 0.16);
}

Inputs & Outputs

Inputs

Name Type Usage
labelstringLabel text shown above/left of the select.
placeholderstringPlaceholder text shown when no selection.
optionsSelectOption[]Option list with label/value/disabled.
valueunknown | unknown[]Current selected value(s).
multiplebooleanEnable multi-select.
showFilterbooleanShow filter input within the dropdown.
filterPlaceholderstringPlaceholder for filter input.
labelPositiontop | left | hiddenControls label placement.
requiredbooleanShows required asterisk on the label.
disabledbooleanDisables the select.
showClearbooleanShow clear icon when a value exists.
displayAsChipsbooleanRender selected values as chips.
chipRemovablebooleanShow remove icon on chips.
panelClassstring | string[]Add custom class(es) to the dropdown panel.
optionDisabledboolean | ((option) => boolean)Disable options conditionally.
compareWith(a, b) => booleanCustom value comparison for objects.
idstringCustom id for the control.
namestringCustom name attribute for forms.

Outputs

Name Type Usage
valueChangeEventEmitter<unknown | unknown[]>Emits when selection changes.
clearedEventEmitter<void>Emits when clear icon is clicked.
